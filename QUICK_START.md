## 🛡️ Sécurité GitHub (v2.1)

### ⚡ Actions Rapides
```bash
# Lancer un scan manuel
git push origin main  # Déclenche CodeQL automatiquement

# Vérifier les alertes
gh run list --workflow=codeql.yml

# Voir le dashboard sécurité
open https://github.com/0xCyberLiTech/0xCyberLiTech.github.io/security
```

### 📋 Checklist Sécurité
- [ ] CodeQL activé : **Settings** → **Code security and analysis**
- [ ] Dependabot alerts : ✅ Enabled
- [ ] Secret scanning : ✅ Enabled (auto pour dépôts publics)
- [ ] Security policy : ✅ [SECURITY.md](SECURITY.md) créé
- [ ] Documentation : ✅ [Guide complet](docs/SECURITY_GUIDE.md) disponible
