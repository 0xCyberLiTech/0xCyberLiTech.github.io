# partials

Ce dossier contient les fichiers partiels HTML et JavaScript utilisés pour composer les différentes parties réutilisables du site web, comme l'en-tête, le pied de page, les modales, etc.

## Contenu
- `header.html` : En-tête du site
- `footer.html` : Pied de page du site
- `modal.html` : Modale générique
- `inject-partials.js` : Script pour injecter dynamiquement les partiels HTML
- `modal.js` : Script pour la gestion des modales

N'hésitez pas à ajouter d'autres partiels pour améliorer la modularité du site.
